retwis-pytor
===========

**retwis-pytor** is a *Twitter clone* based on [Redis](http://code.google.com/p/redis/) utilizing the [Tornado](http://www.tornadoweb.org) framework.

Usage
----

    >>> python ./retwis.py

Dependencies
------------
    * Redis (redis.io)
    * Tornado (tornadoweb.org)
    * Redis-Py (https://github.com/andymccurdy/redis-py)
